# Learning Track: AI-Native SDLC

## Track Title
AI-Native Software Development Lifecycle (SDLC)

## Purpose Statement
Transform the entire software development lifecycle with AI. From requirements gathering to deployment, learn how to embed AI into every phase of building software.

## Who Is This For?

**Primary Audience**:
- Software Engineers (all levels)
- Tech Leads
- Engineering Managers
- Full-Stack Developers
- DevOps Engineers

**Secondary Audience**:
- Product Managers (technical)
- QA Engineers
- Solutions Architects

## Key Outcomes

Upon completion, you will be able to:

1. **Apply AI to every SDLC phase**: Requirements, design, development, testing, deployment, and monitoring
2. **Increase development velocity**: Use AI to reduce time on repetitive tasks by 40-60%
3. **Improve code quality**: Leverage AI for code review, testing, and documentation
4. **Build AI-assisted workflows**: Create repeatable patterns for your team

## Prerequisites

- Navigate (Foundation) track completion
- 1+ years of software development experience
- Familiarity with at least one programming language

## Hands-On Project

### Capstone: AI-Native Feature Development

Complete an end-to-end feature development using AI throughout:

1. **Requirements**: Generate and refine requirements with AI
2. **Design**: Create technical design with AI assistance
3. **Development**: Build the feature using AI coding tools
4. **Testing**: Generate and execute tests with AI
5. **Documentation**: Document the feature with AI
6. **Review**: Conduct AI-assisted code review

**Deliverable**: Working feature + portfolio documentation showing AI usage at each phase

**Verification**: Automated code checking + peer review
