# Module 8 Capstone Project: Multi-Modal Assistant

> **Duration**: 6-8 hours | **Type**: Comprehensive Assessment
> **Submission**: Working Application + Documentation

---

## Project Overview

Build a multi-modal assistant that processes images, audio, and text.

---

## Requirements

### Part 1: Core Pipeline (2 hours)

- Voice input/output with Whisper and TTS
- Image understanding with GPT-4o Vision
- Text processing with conversation memory

### Part 2: Vision Features (2 hours)

- Document/receipt analysis
- Object detection and description
- Multi-image comparison

### Part 3: Audio Features (2 hours)

- Meeting transcription and summarization
- Voice-controlled commands
- Audio sentiment analysis

### Part 4: Integration (2 hours)

- Unified interface handling all modalities
- Graceful degradation
- Caching and optimization
