# Unit 1: Agent Fundamentals

> **Unit Duration**: 4 hours | **Lessons**: 4
> **Module**: 4 - Agents and Tool Use

---

## Unit Overview

Before building agents, you need to understand what they are, how they work, and when they're appropriate. This unit provides that foundation.

---

## Learning Objectives

1. Define what makes an AI agent
2. Understand the ReAct reasoning pattern
3. Compare agent architectures
4. Recognize when simpler solutions are better
