# Module 7 Capstone Project: Secure LLM System

> **Duration**: 6-8 hours | **Type**: Comprehensive Assessment
> **Submission**: Hardened System + Security Documentation

---

## Project Overview

Harden an LLM application with comprehensive security controls, monitoring, and incident response.

---

## Requirements

### Part 1: Threat Assessment (1 hour)

- Complete threat model (STRIDE)
- Risk assessment matrix
- Prioritized mitigations

### Part 2: Input/Output Security (2 hours)

- Multi-layer input validation
- Pattern detection
- Output sanitization
- PII protection

### Part 3: Defense Implementation (2 hours)

- Sandwich defense pattern
- Canary tokens
- Rate limiting
- Access control

### Part 4: Monitoring (2 hours)

- Security event logging
- Alert rules
- Dashboard design

### Part 5: Incident Response (1 hour)

- Response playbooks
- Runbook documentation
- Recovery procedures
