# Unit 2: Advanced Prompting Techniques

> **Unit Duration**: 5 hours | **Lessons**: 5
> **Module**: 2 - Prompt Engineering Mastery

---

## Unit Overview

Move beyond basic prompting to powerful techniques that unlock new capabilities. This unit covers patterns that make LLMs more reliable, more capable, and more useful.

---

## Learning Objectives

By completing this unit, you will be able to:

1. Apply few-shot learning effectively
2. Implement chain-of-thought reasoning
3. Use self-consistency to improve reliability
4. Leverage personas for different response styles
5. Chain prompts for complex workflows
