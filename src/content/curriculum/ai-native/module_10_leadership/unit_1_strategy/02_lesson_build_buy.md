### Decision Framework

| Factor | Build | Buy |
|--------|-------|-----|
| Competitive advantage | Core differentiator | Commodity capability |
| Data sensitivity | Highly sensitive | Standard data |
| Customization | Unique requirements | Standard use case |
| Speed | Can wait | Need now |
| Talent | Have AI team | Limited expertise |

### Hybrid Approach

Often the best answer:
```
[Foundation] API providers (OpenAI, Anthropic)
     +
[Customization] Fine-tuned models, RAG
     +
[Integration] Custom tooling and workflows
```

### Total Cost of Ownership

Build costs:
- Team salaries
- Infrastructure
- Training data
- Maintenance

Buy costs:
- Licensing fees
- Usage costs
- Integration effort
- Vendor lock-in risk
