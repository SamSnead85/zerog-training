# Unit 2: Vector Databases

> **Unit Duration**: 4 hours | **Lessons**: 4
> **Module**: 3 - Retrieval-Augmented Generation

---

## Unit Overview

Vector databases are purpose-built for storing and searching embeddings at scale. This unit covers the landscape, managed services, open-source options, and performance optimization.

---

## Learning Objectives

1. Choose the right vector database for your use case
2. Implement vector storage with Pinecone
3. Use open-source alternatives (Chroma, Qdrant)
4. Optimize indexing for performance
