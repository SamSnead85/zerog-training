# Module 3: Retrieval-Augmented Generation (RAG)

> **Duration**: 18 hours | **Certification Path**: All Paths (Required)
> **Prerequisites**: Module 2 - Prompt Engineering Mastery

---

## Module Overview

RAG is the most important pattern for production AI. It combines the reasoning power of LLMs with accurate, up-to-date information retrieval. This module takes you from RAG fundamentals to production-grade implementations.

---

## Learning Objectives

By completing this module, you will be able to:

1. Explain how RAG solves LLM limitations
2. Create and manage vector embeddings
3. Build and query vector databases
4. Implement retrieval strategies that maximize quality
5. Architect production RAG systems
6. Evaluate and optimize RAG performance
